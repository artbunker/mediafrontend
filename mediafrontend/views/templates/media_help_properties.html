Media properties
Property Description
Focus Used to adjust which part of a non-square visual medium's thumbnail is visible.
Upload date The actual upload date of the media.

This is set only when uploading new media or when replacing the files of existing media.
Creation date A user-specified date meant to represent when media were created or published.

Used as the default sort order of media in normal search results. This can be set to any date, and is mostly meant to be used to backdate older media that weren't uploaded until after their creation.

When uploading new media this is set to the current date unless a creation date is specified in the upload form.

You can nudge the creation date by entering + or - and then an amount of time to adjust the current creation date by, e.g. +1d3h22s. Valid increments are y (years), w (weeks), d (days), h (hours), m (minutes), s (seconds).
Owner The user who currently has ownership of a media.

{# Contibutors can transfer ownership of their own media to other contributors, and moderators can change the ownership of any media. #} Moderators can change the ownership of any media.
Searchability While media will always be returned in search results for moderators (and for the owner of the media within their own media search), you may limit how media are returned in search results for other users.

Media with searchability "Hidden" will not be returned in search results (but will be visible and linked to in media sets below other media in the same sets).
Media with searchability "Hidden" will appear semi-transparent by default.

Media with searchability "Groups" will only be returned for users that have all the required group permissions.
If a medium has no groups selected it will only be returned in search results for signed in users.

Media with searchbility "Public" will be returned in normal search results for anyone.
Protection While media are always viewable to moderators and the owner of the media, you may assign various protection levels which prevent viewing of the medium file, generated summaries, and thumbnails by other users.

Media with protection "Private" are not viewable.

Media with protection "Groups" are only viewable by users that have all the required group permissions.
If a medium has no groups selected it will be viewable to signed in users but users who aren't signed in will be promped to sign in.

Media with protection "None" are viewable by anyone.
Groups The group permissions used in combination with media visibility and protection.

If searchability is public and protection is none but some groups are selected then search result thumbnails will display the selected group tiles, but both the view page and direct links will be viewable if visited.
